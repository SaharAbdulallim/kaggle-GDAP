### Kaggle compettion 
Link : https://www.kaggle.com/competitions/beyond-visible-spectrum-ai-for-agriculture-2026/overview

## Download dataset 

First export your kaggle token

```bash 
export KAGGLE_API_TOKEN=
```

Download 


```bash
kaggle competitions download -c beyond-visible-spectrum-ai-for-agriculture-2026
```

make sure you put the data in /data folder , it should have train and val folder inside /data
